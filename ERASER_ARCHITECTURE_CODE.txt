================================================================================
  FLOW DESCRIPTIONS
================================================================================

1. USER ACCESS FLOW:
   User → Web Browser → Auth → 2FA (admin only) → RBAC → System

2. PROCUREMENT-TO-SALE PIPELINE:
   Supplier → Procurement → Drying → Milling → Stock Distribution
   → Inventory → POS/Orders → Payment

3. ORDER LIFECYCLE:
   Pending → Processing → Shipped → Delivered (or Voided)
   Delivered → Return Requested → Picking Up → Returned

4. DELIVERY FLOW:
   Order Mgmt → Delivery Mgmt ← Driver updates
   Pending → In Transit → Delivered/Failed

5. COST TRACING FLOW:
   Procurement Cost + Drying Cost → Processing → Stock Distribution
   → Cost Per Unit → Profit Margin → Dashboard Analytics

6. ADMIN VS SUPER ADMIN:
   Admin: Core Modules (green) only
   Super Admin: Core Modules (green) + Super Admin Modules (purple)

7. SECURITY CHAIN:
   Request → Token Check → 2FA (if admin) → Role Check → Audit Log → Access
