SYNTAX RULES USED:
  Node [shape: oval]                → oval/terminal shape (start/end)
  Node [shape: diamond]            → decision/condition shape
  Node [shape: rectangle]          → process/action shape
  Node [icon: name]                → feather icon
  Node [color: red]                → node color
  A > B                            → arrow from A to B
  A > B: label                     → arrow with label text
  // comment                       → comment line (ignored)

FLOWCHART COVERS:
  1. User Authentication & Role Routing
  2. Supply Chain: Procurement → Drying → Processing → Stock Distribution
  3. Sales: POS Walk-In & Online Orders
  4. Order Status Flow: Pending → Processing → Shipped → Delivered
  5. Return Flow: Request → Review → Pickup → Returned → Restock
  6. Void Sale Flow with Super Admin Authorization
  7. Analytics & Dashboard
  8. Super Admin Exclusive Features
  9. Audit Trail Logging (all actions)
